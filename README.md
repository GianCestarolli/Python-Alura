### Principais comandos VENV
#### Criando o ambiente virtual:
- "python -m venv venv"
#### Ativando/desativando o ambiente virtual:
- "./venv/Scripts/activate"
-  "deactivate"
#### Criando o "Requirements.txt":
- "pip freeze > requirements.txt"
